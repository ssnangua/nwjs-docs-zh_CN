# 特性列表 {: .doctitle}
---

> 原文链接：https://github.com/nwjs/nw.js/wiki/Features-list

以下为 NW.js 支持的特性列表 _(包含但不限于)_。如果您正在考虑是否要选择 NW.js 来开发应用，本页面可以作为一个参考。

* 全面支持在浏览器中运行 node.js
* 易于使用
* 本地 UI 库
* 无边框窗口
* 打包与发布
* 兼容 npm
* 调试器支持
* 丰富的文档
* Kiosk 模式（类似于 PPT 文稿演示时的全屏模式）
* 文件对话框
* 媒体（图片、音频、视频等）
* 良好的 HTML5 支持：
    * Web 组件
    * 拖放
    * 数据持久性
    * WebGL
    * WebRTC
    * datalist
    * CSS3
